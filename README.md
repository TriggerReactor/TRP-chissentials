### TRP(Trigger Reactor Projects) 으로 분류
  이 프로젝트는 github상에서 Trigger Reactor 을 사용한 프로젝트로 분류되어 TriggerReactor Organization 의 멤버에 의해 Fork되었습니다.
  프로젝트 제작자는 해당 조치에 대하여 철회를 요구할 권리를 가집니다.
